## 文档结构定位指引

### 招股说明书 / 募集说明书 / 重组报告书 概览章节定位

| 内容 | 北交所 BSE | 上交所 SSE | 深交所 SZSE |
|------|-----------|-----------|-----------|
| 概览标题 | 第二节 概览 | 第二节 概览 | 第二节 概览 |
| 主营业务 | 第五节 "发行人主营业务情况" | 第二节 "发行人主营业务经营情况" | 第二节 "发行人主营业务经营情况" |
| 核心财务 | "主要财务数据和财务指标" | "发行人报告期的主要财务数据和财务指标" | "发行人报告期内的主要财务数据和财务指标" |
| 财务单位 | 元 | 万元 | 万元 / 亿元 |
| 风险因素 | 第三节 风险因素 | 第三节 风险因素 | 第三节 风险因素 |

### 必须抽取的财务指标

| 指标 | 中文 | 格式 |
|------|------|------|
| 营业收入 | 营业收入 | X.XX 亿 或 XXXX 万 |
| 归母净利润 | 净利润 / 归属于母公司股东的净利润 | X.XX 亿 或 XXXX 万 |
| 毛利率 | 综合毛利率 | XX.XX% |
| 扣非净利润 | 扣除非经常性损益后净利润 | X.XX 亿 或 XXXX 万 |
| 净资产收益率 | 加权平均净资产收益率 | XX.XX% |
| 经营现金流 | 经营活动现金流量净额 | X.XX 亿 或 XXXX 万 |
